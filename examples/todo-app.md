# Task
간단한 Todo 앱을 만든다. 추가 / 완료 / 삭제가 가능한 로컬 전용 웹 페이지.

# Context
- 런타임: Node 20+
- 프레임워크 자유, 최소 디펜던시 선호
- 단일 HTML 파일로 동작해도 OK

# Constraints
- 외부 API / DB 없음. localStorage 만 사용.
- 접근성: 키보드만으로 모든 동작 가능해야 함.

# Success Criteria
- `npx serve .` 또는 파일 직접 열기로 동작
- 새로고침해도 목록이 유지됨
- 체크박스로 완료 표시, Delete 버튼으로 삭제
